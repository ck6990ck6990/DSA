# DSA

Problem definition:
    Calculate inverse of a square matrix by Gaussian elimination.
    
---------------------------------
Input format:\n
n\n
a11 a12 ... a1n\n
a21 a22 ... a2n\n
.\n
.\n
an1 an2 ... ann\n

n is the dimension of matrix.\n
a is the input matrix.\n
----------------------------------
Output:\n
error\n
b11 b12 ... b1n\n
b21 b22 ... b2n\n
.\n
.\n
bn1 bn2 ... bnn\n

error is the difference between (a*b) and I.\n
b is the inverse matrix.\n
