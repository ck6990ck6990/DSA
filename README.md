# DSA

Problem definition:
    Calculate inverse of a square matrix by Gaussian elimination.
    
---------------------------------
Input format:
n
a11 a12 ... a1n
a21 a22 ... a2n
.
.
an1 an2 ... ann

n is the dimension of matrix.
a is the input matrix.
----------------------------------
Output:
error
b11 b12 ... b1n
b21 b22 ... b2n
.
.
bn1 bn2 ... bnn

error is the difference between (a*b) and I.
b is the inverse matrix.
